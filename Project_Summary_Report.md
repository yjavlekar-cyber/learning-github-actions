### **Mistake 2: Missing local Images**
*   **Issue:** Website was showing broken image icons.
*   **Cause:** We hadn't copied the images from the `Downloads` folder into the `frontend/Images` directory inside the project.
*   **Correction:** Consistently used `copy` commands to move all `_DSC` and `IMG` files into the project structure.

### **Mistake 3: The 'ContainerConfig' KeyError**
*   **Issue:** `docker-compose up` failed with `KeyError: 'ContainerConfig'`.
*   **Cause:** Incompatibility between the old Python-based `docker-compose` (v1.29) and the modern Docker engine when trying to recreate existing containers.
*   **Correction (The "Clean Slate" Method):**
    1.  `docker stop $(docker ps -q)` -> Stops all containers.
    2.  `docker rm $(docker ps -a -q)` -> Deletes old container states.
    3.  `docker-compose up --build` -> Fresh start without legacy state issues.

### **Mistake 4: Shell Syntax Errors**
*   **Issue:** "Command injection detected" or "Terminator expected" during web scraping.
*   **Cause:** Attempting complex PowerShell loops or nested quotes in a single line.
*   **Correction:** Broke commands down into individual `Invoke-WebRequest` and `Select-String` steps.
